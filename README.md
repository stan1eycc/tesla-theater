# Stan's Tesla Theater ⚡

個人專屬的 Tesla 全螢幕串流啟動器。

## 使用方式

1. 在 Tesla 車載瀏覽器開啟：`https://<username>.github.io/tesla-theater/`
2. 點「啟動全螢幕」進入全螢幕模式
3. 選擇你要的串流服務

## 原理

利用 YouTube redirect (`youtube.com/redirect?q=<url>`) 讓 Tesla 瀏覽器以全螢幕模式開啟目標網站。

## 自訂

編輯 `index.html` 中的 card 區塊即可新增/移除服務。

## 注意

- P 檔閃雙黃燈時無法使用全螢幕
- 需要各串流平台的有效帳號
